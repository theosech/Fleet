Detailed Description

This represents the state of a partial evaluation of a program, corresponding to the value of all of the stacks of various types (which are stored as templates from VM_TYPES).
The idea here is that we want to be able to encapsulate everything about the evaluation of a tree so that we can stop it in the middle and resume later, as is required for stochastics. This must be templated because it depends on the types in the grammar. These will typically be stored in a VirtualMachinePool and not called directly, unless you know that there are no stochastics.
