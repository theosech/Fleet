thread_local_rng rng
 

Detailed Description

This is a thread_local rng that is seeded on construction so that each thread can get a different seed. (Because this seeding is in a constructor, it will be called before rlRng is used) If we just make a thread_local variable, it is not seeded well. Note that if we specify random_seed it only applies to the first thread (and you get a warning for other threads)
